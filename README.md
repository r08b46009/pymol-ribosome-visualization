# PyMOL Ribosome Visualization

This repository contains PyMOL scripts for visualizing ribosomal RNA A-helices, structural motifs, and reproducing analysis figures.

## Contents

- `scripts/` — PyMOL scripts for structure visualization
- `examples/` — example rendered images or outputs

## Usage

### Requirements
- PyMOL
- Corresponding ribosome structure files (PDB)

### Running a script
1. Open PyMOL.
2. Load the corresponding structure file.
3. Run a script in the PyMOL command line:

```text
@script_name.pml
```
